### Go的并发

在Go中，独立运行的任务被称为**goroutine**。

启动goroutine，就是调用函数，区别在于调用函数之前写下一个go。如下面的例子所示：

``` go
package main

import (
	"fmt"
	"time"
)

func main() {
	go sleepyGopher()   // 启动goroutine
	time.Sleep(4 * time.Second)
}

func sleepyGopher() {
	fmt.Println("Gopher sleep ...")
	time.Sleep(3 * time.Second)
}
```
