### 创建一个简单的Node.js应用

新建app.js

```javascript

```

新建Dockerfile

```
FROM node:7
ADD app.js /app.js
ENTRYPOINT ["node", "app.js"]
```

运行如下命令生成镜像

```shell
docker build -t kubia .
docker images
```

运行容器镜像

```shell
docker run --name kubia-container -p 8080:8080 -d kubia
```

访问应用

```shell
curl localhost:8080
"Failed connect to localhost:8080; 拒接连接"
```

这里怎么调试下？

使用docker logs查看日志

```
docker logs c957f1db3ff6
```

修改app.js, 重新制作镜像即可

```shell
docker rm c957f1db3ff6
docker imags
docker rmi kubia
```
