### 使用Rancher搭建K8S集群

#### 运行Rancher

1. 准备一台64位的Linux主机，至少4GB内存，安装docker version>18.09.2

   ```
   sudo apt install docker.io
   ```

2. 运行如下命令

   ```shell
   sudo docker run -d --restart=unless-stopped -p 80:80 -p 443:443 rancher/rancher:stable
   ```

3. 访问https://<server_ip>

   设置admin密码
