## 2019/10/11





### Redis安装

下载最新版本[redis-5.0.5.tar.gz](http://download.redis.io/releases/redis-5.0.5.tar.gz)

```
tar xzf redis-5.0.5.tar.gz
cd redis-5.0.5
cd src
make
./redis-server ../redis.conf
```

### Hello Redis

```
./redis-cli
127.0.0.1:6379> set hello world
127.0.0.1:6379> get hello
```
